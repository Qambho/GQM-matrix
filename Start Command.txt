GQM Astro-Quant — Start Commands
================================

First time setup:
  cd c:\Users\HP\Documents\GitHub\GQM-matrix
  python -m venv .venv
  .\.venv\Scripts\Activate.ps1
  pip install -e .

Start server:
  cd c:\Users\HP\Documents\GitHub\GQM-matrix
  .\.venv\Scripts\Activate.ps1
  python -m gqm_matrix.server

Or without activating venv:
  .\.venv\Scripts\python.exe -m gqm_matrix.server

Open in browser:
  http://127.0.0.1:8000

Stop server:
  Ctrl + C

If port 8000 is in use:
  Get-Process python* -ErrorAction SilentlyContinue | Stop-Process -Force
  python -m gqm_matrix.server
